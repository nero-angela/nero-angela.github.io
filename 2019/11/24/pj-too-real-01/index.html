딥러닝을 이용하여 더욱 현실감 있는 AR 앱 만들기 - 1부

Nov 24, 2019 in Project

딥러닝을 이용하여 더욱 현실감 있는 AR 앱을 개발한 이야기를 4부에 걸쳐서 공유합니다.

목차

  • ✔︎ 1부. 딥러닝을 이용하여 더욱 현실감 있는 AR 앱 만들기
  • 2부. ARKit과 SceneKit을 이용하여 iOS 앱 만들기 - 작성중
  • 3부. YOLO v3를 이용하여 조명 인식 딥러닝 모델 만들기 - 작성중
  • 4부. CoreML을 이용하여 iOS 앱에 딥러닝 모델 올리기 - 작성중

1. 프로젝트 소개

시작하기에 앞서 프로젝트에 사용된 어플리케이션은 Apple에서 제공하는 Sample Code인 Placing Objects and Handling 3D Interaction의 일부 코드를 이용하여 만들었음을 밝힙니다.

1.1 프로젝트 목표

AR(Augmented Reality)을 이용한 다양한 서비스들이 출시되고 있습니다.

IKEA에서 출시한 AR 어플리케이션 Place
출처 : https://mobile-ar.reality.news

대부분의 AR에 사용되는 3D 모델들은 아래 Apple Sample Code인 Placing Objects and Handling 3D Interaction에 사용된 램프 모델과 같이 고정된 그림자 이미지를 가지고 있습니다.

Apple의 Sample Code에 사용된 램프 모델
출처 - Placing Objects and Handling 3D Interaction

✔︎ 문제인식
현실 조명의 위치와 밝기 정보가 반영되지 않아 가상 오브젝트의 질감과 그림자가 주변 물체와 이질감이 느껴짐

대부분 어떠한 상황에서도 어색하지 않을만한 범용적인 형태의 그림자를 가지고 있지만, 주변 물체들의 그림자와 다른 방향과 색상으로 인해 현실감이 떨어지기 때문에 다음과 같은 목표를 설정하였습니다.

✔︎ 프로젝트 목표
현실의 조명의 위치와 밝기 정보를 반영한 AR 어플리케이션을 만들자.

2. 프로젝트 결과물

전체 개발 기간은 3개월이 소요되었고, 5개 class를 실시간으로 추적하고 구분하는 Object Detection 모델인 YOLO v3를 학습시켜 iOS 기반의 AR 어플리케이션에 탑재하였습니다.

✔︎ 개발 환경
IDE : XCode / Jupyter Notebook
iOS : ARKit / SceneKit / Vision / CoreML
Object Detection : YOLO v3 / Keras / Tensorflow

iOS 앱은 App Store에 게시해 두었습니다.

2.1 기능소개

백문이 불여일견이니 먼저 어플리케이션 작동 영상을 보시겠습니다.

현실 공간을 스캔하고, 가상의 3D Object를 설치하는것 까지는 IKEA의 PLACE 앱과 동일합니다.
하지만 추가적으로 YOLO v3라는 실시간 물체추적(Object Detection) 딥러닝 모델을 이용하여 현실에 있는 조명의 상태와 위치를 실시간으로 파악하여 가상 오브젝트의 그림자와 질감을 더욱 현실감 있게 만들어주는 기능을 추가하였습니다.

전체적인 기능을 나열하면 다음과 같습니다.

✔︎ 기능명세

  • 벽과 바닥 탐지
  • 벽과 바닥에 설치가능한 제품 추천
  • 가상 오브젝트 설치 및 인터렉션
  • 5개 class의 실시간 조명 탐지(Bounding Box, Confidence)
  • 범용적인 조명 탐지를 위한 Google Vision API 연동

앞으로 4부에 걸쳐 개발 후기를 공유하도록 하겠습니다.
